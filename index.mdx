## What is E-Dokan?

E-Dokan (meaning "E-Shop") is a storefront-only Next.js application designed to be paired with a headless commerce backend that exposes a REST API. The store fetches all product, category, billboard, size, and color data at runtime from an external API configured via an environment variable.

Key characteristics:

- **Server-side rendering by default** — product and category pages are rendered on the server for fast initial loads
- **Client-side cart** — the shopping cart lives entirely in the browser using Zustand, persisted to `localStorage`
- **No shopper auth required** — no login needed to browse or add items to cart
- **Responsive** — works across mobile, tablet, and desktop screens
