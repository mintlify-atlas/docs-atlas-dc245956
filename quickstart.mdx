## Prerequisites

Before you begin, make sure you have the following:

- **Node.js 18 or later** — [Download here](https://nodejs.org)
- **A package manager** — npm (bundled with Node.js), [yarn](https://yarnpkg.com), or [pnpm](https://pnpm.io)
- **A running backend API** — E-Dokan Store is a storefront-only app. It requires a headless backend that exposes the REST endpoints listed below.

<Note>
  E-Dokan Store does not include a backend. It fetches all data at runtime from
  an external API configured via the `NEXT_PUBLIC_API_URL` environment variable.
  Your backend must serve the following endpoints:

  - `GET /products`
  - `GET /categories`
  - `GET /billboards/:id`
  - `GET /sizes`
  - `GET /colors`
</Note>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/hempun10/edokan-store.git
    cd edokan-store
    ```
  </Step>

  <Step title="Install dependencies">
    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Configure your environment">
    Create a `.env.local` file in the project root and set your backend API URL:

    ```bash .env.local
    NEXT_PUBLIC_API_URL=https://your-api-url.com
    ```

    Replace `https://your-api-url.com` with the base URL of your running backend. All API requests in the storefront are prefixed with this value.
  </Step>

  <Step title="Start the development server">
    <CodeGroup>

    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    ```bash pnpm
    pnpm dev
    ```

    </CodeGroup>

    This runs `next dev` and starts the Next.js development server with hot reloading enabled.
  </Step>

  <Step title="Open the storefront">
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser. You should see the homepage with the featured billboard and product list, both fetched live from your backend API.
  </Step>
</Steps>
